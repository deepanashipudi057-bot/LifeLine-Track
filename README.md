# LifeLine Track

Medical-themed SaaS landing page: health dashboard with sidebar navigation, red/white/light blue theme, and responsive layout.

## Contents

- `index.html` – Main page (sidebar + hero, features, pricing, CTA, footer)
- `styles.css` – Medical theme, Inter/Poppins, rounded cards, soft shadows
- `script.js` – Sidebar toggle (mobile), smooth scroll

## Run locally

Open `index.html` in a browser or use a local server.
